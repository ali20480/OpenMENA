# OpenMENA

This repository contains the supplementary material of the paper "OpenMENA: An Open-Source Memristor Interfacing and Compute Board for Neuromorphic Edge-AI Applications" by Ali Safa et al., College of Science and Engineering, Hamad Bin Khalifa University, Doha, Qatar.

If you are using this work, please cite as follows:

Ali Safa, Farida Mohsen, Zainab Ali, Bo Wang, & Amine Bermak. (2025). OpenMENA: An Open-Source Memristor Interfacing and Compute Board for Neuromorphic Edge-AI Applications.

The pre-print of the paper can be found in the link below:

https://arxiv.org/abs/2511.03747
